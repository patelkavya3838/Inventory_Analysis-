## Inventory Optimization & Stock Analysis

### Project Overview
Built an inventory optimization analysis in Excel using 
Power Query and Data Model covering ABC classification, 
safety stock, supplier performance, and obsolescence risk
across 300 parts and 280k rows of supply chain data.

### Tools Used
- Microsoft Excel
- Power Query
- Power Pivot (Data Model)

### Analysis Performed
- ABC Classification
- Stock Health Analysis
- Safety Stock & Reorder Points
- Supplier Performance Scoring
- Slow Moving & Obsolescence Risk

### Data Source
Data obtained from Kaggle — https://www.kaggle.com/datasets/robertocarlost/aerospace-supply-chain-performance-and-forecasting?utm_source=chatgpt.com 
for educational and portfolio purposes only.

### Key Findings
- A-class parts represent 48.68% of total inventory value
- A-class parts are experiencing frequent stockouts
- 7 parts sourced from high risk suppliers
- Multiple parts identified as slow moving with shelf life risk
